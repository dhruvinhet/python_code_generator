About This Visualizer

Welcome to the AVL Tree Visualizer! This tool allows you to input a sequence of numbers and watch as they are inserted into an AVL tree. The AVL tree is a self-balancing binary search tree, which maintains its balance factor (the difference in height between left and right subtrees) to be -1, 0, or 1. This ensures that tree operations (insertion, deletion, search) remain efficient, even for large datasets.

To get started, simply enter numbers into the input field below, separated by commas. Click "Build Tree" to visualize the resulting AVL tree.
